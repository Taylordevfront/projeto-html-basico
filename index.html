Este é um parágrafo de exemplo que contém várias formatações em itálico, sublinhado, destaque, texto pequeno, e até mesmo números como H2O ou 210.

Podemos também mostrar o uso de texto riscado, que pode ser útil para mostrar alterações. A tag font (embora obsoleta) pode mudar cor e estilo da letra, mas hoje em dia usamos CSS para isso.

Forçando uma
quebra de linha dentro do texto para exemplificar a tag <br>.

HTML é a linguagem padrão para a criação de páginas web. Com ele podemos estruturar títulos, parágrafos, listas, links e muito mais.

Além disso, é possível criar documentos acessíveis e organizados. Por exemplo, se quisermos destacar algo importante, podemos usar marcação em amarelo.

Já se quisermos adicionar uma observação discreta, podemos usar o texto pequeno.

Lista Ordenada - Passos para Criar uma Página

  1. Abrir um editor de texto (ex: VS Code)
  2. Escrever o código HTML básico
  3. Salvar o arquivo com extensão .html
  4. Abrir o arquivo no navegador

Lista Não Ordenada - Tecnologias Relacionadas

  • HTML - Estrutura
  • CSS - Estilos
  • JavaScript - Interatividade
